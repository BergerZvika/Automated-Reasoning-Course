PySMT is a Python library providing a unified API to SMT solvers (Z3, MathSAT, CVC4…).  ·  Build formulas with Symbol, And, Or, Plus, Equals, etc.  ·  Solve with Solver or is_sat and read models with get_value.  ·  Browser execution powered by Pyodide + Z3 WebAssembly — first run downloads ~30 MB.

Symbol(name, INT)
Creates a typed integer variable. Use INT, REAL, or BOOL from pysmt.typing.
LE(a, b) / GE(a, b)
Less-than-or-equal / Greater-than-or-equal constraints.
And(...) / Plus(...)
And conjoins formulas; Plus sums terms. Both accept Python generators.
Equals(a, b)
Equality constraint — use this instead of Python's == which compares objects.
Solver(logic="QF_LIA")
Context-manager solver. QF_LIA = Quantifier-Free Linear Integer Arithmetic.
solver.get_value(sym)
Returns a PySMT constant. Use .is_true() for booleans, not Python's == True.

Quantifiers
Universal quantification with ForAll
Check whether ∀x. x+1 < x is satisfiable. This is a contradiction — no integer satisfies it — so the result is UNSAT.
ForAll([vars], formula)
Universal quantifier. First argument is a list of bound variables.
Exists([vars], formula)
Existential quantifier — works the same way as ForAll.
LT(a, b)
Strict less-than. Equivalent to a < b via operator overloading.
Quantified logic
Goes beyond QF (quantifier-free) logics. Not all solvers handle quantifiers — Z3 does.
